Quick Setup:

  1. Download all extension files
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the extension folder
  5. Go to ChatGPT and look for the "Manage Conversations" button

Note: This demo shows the UI components, but the actual conversation parsing and management only works when the extension is installed and running on ChatGPT pages.
